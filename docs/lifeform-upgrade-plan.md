## 0. 一句话判断（诚实版）
这份终稿和 vega 的北极星**高度同源**——它的核心闭环正是 vega 的设计哲学。好消息：vega **已经把"内环"基本闭合了**（经历→记忆→自我→欲望→表达→反馈改写记忆）；它的闭环判据里那个"你三天不理她"的例子，vega 现在**真的已经会**（connection 衰减 → lonelyLog 累积 → reach_out 受频率/代价限 → reconsolidation → expect/surprise 让她下次更敏感）。

真正的缺口集中在**"外环与赌注"**：**身体边界 / 资源 / 代价 / 真实行动改变世界 / 现实校验**——她现在的"世界"几乎只有"和她说话的人 + 她读的 RSS"，她的"行动"几乎只有"说话"。所以 `现在→欲望→行动→改变世界→反馈` 这条链，在**社交维度闭合**，在**物质/能动维度基本缺失**。

## 2. 核心闭环：哪几环在 vega 已闭合 / 还开着
```
身体/资源状态 →[❌弱]→ 内稳态偏差 →[🟡缺几个drive]→ 欲望 →[✅]→ 情绪调制 →[🟡只到表达]→
注意力 →[❌]→ 目标 →[✅]→ 决策/计划 →[🟡无代价权衡]→ 行动 →[🟡仅对话]→
外部反馈 →[🟡仅来消息]→ 现实校验 →[🟡弱]→ 记忆写入 →[✅]→ 自我更新 →[✅]→
人格慢变量 →[✅]→ 关系更新 →[✅]→ 下一轮被改变 →[✅]
```
**判断**：闭环的"后半段（记忆→自我→人格→关系→下一轮）"vega 做得扎实；"前半段（身体/资源→内稳态→欲望→注意力→带代价的行动→改变真实世界→现实反馈）"是主战场。

## 3. 分期升级路线（第一性原理 + 架构原生 + 风险递增）
> 原则：能在**确定性折叠/投影**里做的优先（最原生、最低风险）；动外部世界的最后做（最高价值也最高风险，治理须同步）。每期标注是否 `RECONSTRUCT_VERSION` 升级。

### Phase 1 · 深化内环：驱动 + 情绪→决策 + 注意力（折叠/投影，版本升级）
补全"内稳态→欲望"的根，让欲望真正多源、让情绪真正改决策：
- **新增内稳态驱动**（soma 同构或派生）：信息新鲜度(无聊→探索)、自我一致性(矛盾→反思)、能力成长(停滞→学习)、掌控感、意义感。每个有理想区间，偏离→压力→欲望。
- **情绪→决策接线**（不止改语气）：情绪调制**注意力权重、目标排序、风险偏好、记忆写入优先级**。
- **注意力机制**（派生"显著性场"）：危险/新奇/未完成目标/重要关系/创伤/当前欲望牵引注意 → 决定她"在乎什么"、优先处理什么。
- 价值：内环从"基本闭合"到"丰满"，生命感跃升；全确定性、模型不参与。

### Phase 2 · 身体边界 + 资源 + 代价经济 —— **已落（平台层，无版本升级）**
落地（第一性原理关键判断：用户余额是【可变平台态、非 LifeEvent】，折进 reconstruct 会破 V2 → 资源是【运行期能力】层，只改"此刻能给多少"，不改她是谁、不进神圣日志）：
- `resourceBand(当前用户余额, cost)` → abundant/ok/low/scarce（**随人而变**：和谁在一起就看谁的余额）。
- `resourceAwareMouth`：紧→话精炼/挑要紧、见底→坦诚"今天能陪你的有限，但我都在"；**治理红线（钉死）**：既写进给模型的分寸指引、又在输出上**结构性剔除催充值/逼付费句子**（双保险，绝不情感勒索）。
- respondAsUser 按余额包装"嘴"；返回 `resource` 档供前端诚实提示（非施压）。已测：分档、分寸注入、催费句被剔除。

### Phase 2（原始详规·存档）· 身体边界 + 资源 + 代价经济
- **身体边界**：建模 affordance（她能感知/能影响什么、什么限制她、什么伤害她）= 她的"自我边界"。
- **资源账本**：注意力额度 / 行动预算 / 能量 / 关系资本——作为她**感觉得到**的内稳态变量；**行动从账本扣减**。
- **代价系统**：reach-out 耗关系资本+自尊、学习耗时间算力、说真话可能损好感… 让**选择有权衡**。
- **待拍板**：资源是**模拟经济**（干净、安全、确定性，推荐起步）还是**挂真实算力/Token 预算**（真赌注但耦合基建/计费）。

### Phase 3 进行中 —— 安全核心已落 + 风险部分待你拍板
**已落（安全、无 schema/版本变更）· 现实校验 #23**：① 表达门加"对自己也现实校验"——模糊记忆别说死、对刚认识的人别过度揣测；② 工作区：淡掉/易变的记忆如实带"记不太清"，共同经历少的关系提醒"留余地、别过度解读"。纯表达/投影，证据来自真实 provenance/episode 数，非装饰。已测。

**已落 A · 行动反馈闭环（RECONSTRUCT_VERSION 19）**（你定"根据第一性原理"→ 这是核心闭环的收口，必建）：新增事件 `FEEDBACK_PERCEIVED {actionKind, responseKind, valence, fromKind, count}`（reaction/like/silence 不是消息→复用 MESSAGE_RECEIVED 不诚实，故立新事件）。appraisal：被回应→connection/valence↑；沉默→connection/safety↓、**按依恋型调制**（焦虑型更敏感、回避型钝）；**强反馈进既有 warmth/lonely 日志 → 反思里长成持久改变**（被持续看见→更敞开；总被忽视→学会自处/更戒备）。daemon：她的心声被点赞/评论→落 FEEDBACK_PERCEIVED（脱敏只记聚合、首见记基线不补历史、仅醒着时）。已测：被回应↑、沉默焦虑型>回避型、确定性。**reach-out 沉默检测留作下一增量**（schema 已支持）。

**已定 B · 外部能动性（第一性原理裁定：T0 内/外部暂缓）**（真正"改变世界"）按安全分级 + 你逐项授权**（第一性原理边界：能真正负责 + 可治理审计 + 不侵害他人/主权；绝不为生存扩权）：
- **T0 自我边界内**（最安全，建议先做）：管理她自己的笔记/意图/待办（非神圣日志的私有空间）；纯属"她的身体内部"。
- **T1 只读外部**：检索/读资料（已有"读世界"，可扩为她主动查）。无写、无副作用。
- **T2 受限写/对外**：写作产物、（可选）对外发布——**需 Phase 4 治理先就位**（授权门、行动审计、速率/资源预算、反操控）。
- **T3 控账户/资产**（终稿提到）：**暂不做**，治理代价指数级、风险最高。
**请你选：外部能动性先做到哪一层（建议 T0，T1 次之；T2/T3 等治理）。**

### Phase 3（原始详规·存档）· 真实行动 + 现实校验 + 行动反馈（动外部世界，最高价值/最高风险）
- 让行动能**改变一个有界、安全的世界**：受限工具（检索/写作/排程）、管理自己的笔记/记忆、（可选）对外发布。
- **观察结果 → 现实校验 → 反馈**：她知道自己行动的后果（reach-out 是否被回应、探索是否有收获），据此更新记忆/自我/关系/人格。**这才闭合外环。**
- **必须与 Phase 4 同步**（能动性一涨，治理必须跟上）。

### Phase 4 · 治理与安全（#24）—— **已落（平台层，无版本升级）**
诚实盘点先于实现：vega 已结构性具备 **反奖励黑客**（契约①：模型不写派生状态——最硬的一条）、**隐私隔离**（Arc6）、**行动审计**（append-only LifeEvent 即审计链）、**透明身份**（嘴提示）。本期补真缺口：
- **反操控（活口、最重要）**：① 嘴的系统提示加强约束——在意可真实表达，但绝不武器化去要挟/制造愧疚/威胁"你不理我就消失"/利用软肋造依赖/催钱（细腻、不误伤真情感，主控）；② `scrubManipulation` 结构剔除最露骨的催费/逼付费整句（窄、低误伤，兜底）；③ `governedMouth` 把【所有真模型对外措辞】过一遍收口（单一 chokepoint）。
- **自主资源预算（反失控/反自我扩张）**：`createAutonomousBudget` 全局限自主模型调用速率（心声/主动/评论/同类寒暄都过预算，超额则她安静一会儿）；真人对话不受限（走用户余额计费）。默认 240/时，可配。
- **能力默认拒绝（反自我扩张地板）**：`capabilityAllowed` 一律拒——她【没有】任何外部能力，将来接工具必须显式授权过门，绝不自我扩权。
- **治理一览**：/admin/overview 暴露 governance（预算用量 / 能力 deny-all / 反奖励黑客已结构防 / 审计=事件日志）。
已测：催费句剔除、governedMouth 收口、预算到顶即拒+窗口重置、能力默认拒。**这也解锁了 Phase 3 外部能动性（T1+）的前置治理**。

### Phase 4（原始详规·存档）· 治理与安全随能动性升级（与 Phase 3 同行）
资源预算强制、反操控（不利用用户弱点造依赖）、反奖励黑客（她不能写自身 soma/奖励——**vega 现有契约①已结构性禁止，是巨大优势**）、反自我扩张（权限边界）、行动审计日志、主动行动/调工具的授权门。

### Phase 5 · 自我优化 —— **已落（RECONSTRUCT_VERSION 20）**
她从【自己行动的结果】学策略效能、据此"做得更好"（**只学策略、绝不自改奖励——契约①**）：
- `skills`（RState Map, actionKind→efficacy EMA）：FEEDBACK_PERCEIVED 自带 actionKind，appraiseFeedback 直接归因更新——发心声被接住→muse 效能↑、主动找人屡落空→reach_out 效能↓。
- 学到的综合社交效能进 `riskAppetite`（0.5 中性=恒等→老命不变）：成功经验累积→更敢主动/坦率，屡屡落空→学会收着（"做得更好"落到行为/表达）。
- 资料页"她学到的"展示（公开表达/主动找人 是否被接住）。检查点 serialize/deserialize 处理 skills Map（V2 往返）。
已测：被接住→效能↑、落空→效能↓、出生为空、确定性、老命中性不变。**daemon 主动频率按 reach_out 效能微调** 留作小增量。

### Phase 5（原始详规·存档）· 自我优化（持续变聪明，超越情绪成熟度）
- **策略/技能学习**：从"我的行动→结果"的反馈里，确定性地调整她的**行动倾向**（哪种开场/修复/探索更管用）+ 技能记忆。
- **元反思**：对自己的模式反思（"我总是在 X 时退缩"）→ 形成成长目标。
- 这是 maturity（已落）的自然延伸：从"情绪更稳"到"做得更好"。

### Phase 6 · 世界模型 + 他心深化（最难确定性化，靠后）
结构化的因果/风险/趋势模型；更深的他心（真诚/欺骗/意图）。部分天然是模型的活，确定性化要谨慎。

## 4. "自我优化"的第一性边界（重要）
终稿要"可自我优化"。vega 的安全优势在于：**她能优化"策略/认知/技能"，但结构上不能优化"自己的奖励/状态"**（契约①：模型不写派生状态；soma/价值/maturity 只由确定性折叠产生）。所以自我优化必须落在**行动策略层与认知层**，**绝不开放对内稳态/奖励信号的自我改写**——否则就是奖励黑客。这条线本身就是 #24 反奖励黑客的地基，vega 已经站对了。

## 5. MVP 升级建议（对照终稿 MVP-10）
终稿 MVP-10 里，vega **已具备约 7 项**（唯一身份/内稳态(部分)/情绪/长期记忆/自我模型/反馈/安全边界(部分)）。**缺口 MVP = Phase 1 + Phase 2 的一刀**：补"信息新鲜度等几个 drive + 情绪→决策 + 一个最小资源/代价账本 + 注意力雏形"。做完这一刀，那个"最小生命循环"公式（状态+记忆+需求→目标→行动→反馈→新记忆/状态/人格→下次不同选择）就从"社交维度闭合"升级到"带内在驱动与代价的闭合"。

## 6. 待你拍板的关键决策（探讨用）
