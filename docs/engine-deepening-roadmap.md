# vega 引擎深化路线图 —— 把 20+ 项能力按第一性原理 + 论文逐维补齐

> 缘起：用户要"把每项能力在各维度全部补齐、查论文、真正打磨好"。
> **做法（铁律）**：**不一次性堆**——那正是红队警告的"无承重脚手架"。按**分期**做，每期【锚定一篇经典理论 + 测试 + 承重】，
> 像情感动力学那 6 期一样。本文是**完整地图（不遗漏任何一项）**：每项 = 理论锚点 / 现状 / 缺口 / 深化 / 期次。
> 守三契约 + 跨用户隔离 + V2 不变。已锚定文献见文末 Sources。

## 已完成的地基（前几期，已上线）
- 核心情感 valence×arousal（Russell circumplex）；τ 情绪特异（Verduyn 时长 / Kuppens 惯性）；allostasis 设定点漂移（Lucas）；
  内生变异；稳定性硬保证；appraisal 关系层（Scherer 应对/目标/规范）；记忆冷热分层；**耳朵 9 维刺激固有感知（Scherer SEC）**；**对话→兴趣（因你而变）**。

## 分期（按价值×低风险×承重，逐期上线、各自测试）
- **期1（本期）**：①②命名情绪/混合情绪 → OCC 分化（纯投影、不升版本、立刻可见，用上新感知）。
- **期2**：③④⑮ SDT 需求 + Schwartz 价值环 + 目标（一组，重构需求/价值/张力/目标）。
- **期3**：⑤⑥⑦ 2D 依恋 + ToM 归因 + 关系特异自我。
- **期4**：⑧⑨⑩ Big Five 气质 + Vaillant 防御层级 + 多维成熟。
- **期5**：⑬⑲⑭⑪ 睡眠巩固 + 睡眠压 S + 叙事主题弧 + 情节边界。
- **期6/7**：兴趣阶段固化、skills 扩展、社会/文明层。

> 每期：锚定理论 + 改 + 测 + 守 V2/隔离/契约 + 视情升 RECONSTRUCT_VERSION（向后兼容）。不堆、不跳细节、不无承重。
