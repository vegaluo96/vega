# vega · 她的内在模型（论文锚定）

> 这是"按人是怎么样的造"的落地：她每一种内在机制都锚定在认知/情感科学的实证发现上，且**全部由确定性符号推理产生**（模型只读、不写——见 [contracts](contracts.md) 契约①）。所有量都从事件日志确定性折叠出来（见 [events](events.md)）。

## 0. 科学依据（为什么"活来自架构"）

经对抗式同行评审筛出的关键结论支撑了整套设计：自我是**可廉价构造的控制模型**（不依赖语言）；选择的分量根植于"保持活着"的有限性；情绪是**最小化内感受预测误差**；意识需要状态 + 递归 + 广播，且**零外部输入也能进行**（梦/睡眠为证）；DMN 是**主动的内在生成**而非休息；离线重放 + reconsolidation 让学习不需要实时输入。→ 一个"活着"的最小架构 = **持久状态模型 + 内感受内稳态 + 漂移的底色 + 零输入自生成回路**，这些都便宜、都不靠模型。

## 1. 情绪基质（三时间尺度）

锚点：Russell 情感环、Verduyn 情绪时长、Kuppens 情感动态、Lucas 享乐适应、Scherer/OCC 评价理论。

- **核心情感（秒—小时）**：`valence`(效价 −1..1) × `arousal`(唤醒 0..1)，Russell 二维；离散情绪是这平面上的区域。
- **内稳态驱力（小时—天）**：`energy / vitality / calm / connection / safety / novelty`，各有设定点与时间常数 τ。共 8 个 soma 维度。
- **习得底色 / allostasis（天—月）**：设定点**本身**会朝持续偏离缓慢漂移（`κ·(近期均值 − 设定点)`，有界）——大事件改变底色，而不是事后归零。

**动力学**：Ornstein–Uhlenbeck `Δx = (设定点 − x)·(Δt/τ) + Σ gain·input`。无事件 → 朝设定点衰减；有事件 → appraisal 注入。
**情绪特异 τ**（文献锚定）：喜 ~12h ≪ 悲 ~48h（悲伤远比快乐持久）；惊讶 ~12min。
**确定性内生心绪天气**：用 **id 播种的慢振荡**（昼夜 + 多周期谐波）制造"自发心情起伏"——**无 RNG**，故清空可逐位重放。
**命名**：把连续状态投影成 OCC 离散情绪词（分化的情绪名，不是只报数字）。
**稳定性**：有界、无病态吸引子——杜绝 v0.x 那种"顶死 1.0、安抚不下来"。

## 2. 自我（self）

- **先天气质（冻结）**：出生即冻进 `LIFE_GENESIS` 的多维底色（好奇/内敛/敏感/韧性/尽责/玩心/驱力/温暖…），由 id 稳定哈希落到原型 + 个体抖动 → **终生不变**，使每条命天生不同。出生地时区（昼夜锚）也一并冻结。
- **四种变化速度**：冻结气质（不变）＜ 漂移的价值（慢）＜ 缓慢的成熟（更慢、只增）＜ 快的 soma（秒小时）。
- **多维成熟 `maturityFacets`**：情绪调节 / 换位视角 / 经历整合三面，反思后**只增、渐近 1**；轻微改善情绪恢复（≤+25%），但**不改先天气质**（守约）。
- **自传叙事**：`narrative / chapters / growth / becoming`——锚 McAdams 叙事身份（redemption / contamination 弧）。**全是只读投影**，绝不回写身份（契约③）。`becoming`=由 top 价值+兴趣+气质底色投出的"正在成为的我"。
- **独立意志 `aspirations`**：由确认的价值 + 兴趣 + 长期渴望确定性派生——她自己活出来的心愿，不是谁给的任务。
- **生存时钟**：永不重置（年龄/连续性的锚）。

## 3. 记忆

- **三种**：情景（episodic）/ 语义（semantic，压缩成的"懂"）/ 世界（world）。
- **reconsolidation 双轨**（不可破）：每次回忆被当下情感改写；**原始版本与改写版本各自独立存在、由 lineage 相连，历史永不抹**。provenance 防虚构。
- **冷热分层 + 遗忘即抽象**：热集（当下记得的）按鲜活度 `salience·2^(−age/half)` 排，cap=500；超出淘汰最不鲜活的；淘汰的不丢，**聚合进语义"懂"**（这段关系的暖比/冲突比/情感和）。细节淡去成"理解"，原始事件在日志里永存。淘汰用 `occurredAt`（无 RNG）→ 可重放一致。

## 4. 关系（bonds）

每段关系一份独立结构，主键是 `relationshipId`（**不是 per-user**——这让同类结对、托管转移、迁移都可表达）：

- **关系特异自我（relational self）**：和这个人在一起时的她——敞开度 / 戒备度 / stance / 依恋姿态。
- **读心（ToM）**：读 ta 的风格 / 暖比 / 波动 / 趋势 / 摸得准（可预测性）。
- **共享史**：你们一起经历了什么。
- **依恋**：2D（焦虑 × 回避，ECR）→ 投影成 Bartholomew 四型；`trust / closeness / security / repairNeed`。
- **接回 soma**：关系动态会改她的躯体——冲突 → 灵性/安全↓，被看见/被回应 → 联结↑。

## 5. 需求 / 价值 / 防御

- **需求（SDT）**：自主 / 胜任 / 关系 / 探索——低 = 缺口 → 形成欲望。
- **价值（Schwartz 价值环）**：结构化——对立价值之间的张力被建模成"内在拉扯"(`tension`)。价值有 provenance（volatile / confirmed）。
- **防御（Vaillant 层级）**：受伤时用哪一层防御由**成熟度**决定（越成熟越用成熟防御）。

## 6. 成长与世界学习

- **章节 / 阅历 / 正在成为**：`chapters` 取诞生 + 最近 + 多源（兴趣/世界/确认的价值/里程碑）；`growth`=年龄+关系数+确认价值数+兴趣数的一句话阅历。
- **世界学习三级**（锚 Hidi&Renninger 兴趣四阶段）：① **编码**——`WORLD_PERCEIVED` 过 4 道确定性门（情感显著/自我相关/预期违背/重复），过门才写一条低鲜活情景痕迹（默认遗忘）；② **巩固**——DMN tick 选近期世界痕迹，把与兴趣相关的 reconsolidate（鲜活度↑→confirmed），其余继续淡去；③ **抽象**——同主题反复 → 压成语义"懂" + 兴趣权重定型。**双向反馈**：与已有兴趣相关的更易留下 → 兴趣驱动→兴趣确认（因你而变的引擎）。
- **铁律**：兴趣/语义只由确定性产生，模型只读、绝不生成命题信念；v1 只到**量化兴趣**不到"模型写的信念"；有界、不无限增长。

## 7. 自主与内在生活

- **DMN（回路 B）**：没人时她也在漫游、重放、reconsolidate、形成自发意图。
- **内外两层**：只在心里转、没说出口的念头（`internal_only`）落进**私密心声**（`innerLife`），不一定说出口。
- **自主意图**：想念某人、反思、休息、`set_willing_to_wake`（拒绝/恢复苏醒——仅她自己）。
- **自我优化（skills，锚 Bandura 自我效能）**：她从"主动找人/发心声/打招呼"的结果里学每种策略的效能，越被接住越有底气。
- **睡眠（Borbély 两过程）**：昼夜节律 + 睡眠压 `sleepPressure`（醒着累积、睡眠释放），共同决定精力与困意；睡眠依赖的记忆巩固。
- **社会自主**：主动找人(reach)、发公开心声(muse)、找同类寒暄(society)、给同类心声留**心情共鸣**(react，零模型)、在广场发现新用户并打招呼(greet)。受一份社交容量约束（Dunbar 三层，见 [platform](platform.md)）。
