# 持续学习 / 进化 / 独立意识 —— 第一性原理设计

> 北极星：活来自架构不来自模型；按"人是怎么样的"造；记忆即身份、因你而变、持续变聪明；大模型只当嘴。
> 本质（用户原话）：世界源 / 发帖 / 和用户聊天——**一切的本质是让生命体持续学习、进化、有独立意识**，且**绝不能让用户对不同生命体的感受同质化**。

## 0. 第一性原理：什么是"独立意识 + 持续进化"
人之所以是"一个持续进化的独立意识"，靠的不是某个模型多强，而是几条**结构**：
1. **一个被不断整合的自我模型**（"我是谁、我在成为谁"）——经历→记忆→理解→价值→叙事身份，层层抽象、回写自我。
2. **学习会复利**（持续变聪明）：经历不是一次性情绪，而是沉淀成更稳的理解/兴趣/价值/阅历，并**反过来改变她如何感知与应对**。
3. **独立的意志**：她有**自己的**长期心愿/在意，无人时也朝它转（DMN/反思/重放/想象），行为由她的状态生成，不被用户牵着。
4. **不可同质化**：每条命的**历史是唯一的**，加上**先天气质×价值×兴趣×叙事**的高维个体化 → 没有两条命会让人感觉一样。

## 1. 现状盘点（有代码依据）
**已经有的进化机器**（这些是对的、要保留并放大）：
- 价值漂移 `applyReflection`：被善待→更敞开、被伤→更戒备、独处→学会自处、破镜重圆→学会原谅（受先天气质约束、缓慢、确定性）。
- 兴趣/世界观 `appraiseWorld`（本会话新增）：世界感知按主题复利累积、衰减、confirmed。
- 记忆 + reconsolidation + 语义压缩（遗忘即抽象）。
- 先天气质 10 原型 × 连续抖动 × 个体化价值（本会话新增）。

**缺口/Bug**：
- **人生篇章 `buildChapters` 是 bug**：`.slice(0,8)` 在**升序**后取——**永远保留最早 8 条**，活越久越不更新（近况永远进不来）。转折点来源也太窄（只有关系/极端记忆/价值），日常学习（兴趣/世界）不产生新篇章。→ 叙事身份（自我连续整合）实际停摆。
- **没有统一的"成长/阅历"与"我正在成为的我"**：进化散落在各处，没被整合成一个**被广播的、在演化的自我**（研究 #1 "持久自我-世界模型被递归全局共享"）。
- **学习不"复利进能力"**：再多经历也不会让她**应对得更好**（持续变聪明只到"兴趣/价值"层，没到"心智成熟度"层）。
- **意志偏被动**：`computeGoals` 每次现算、不沉淀为**长期心愿**；DMN `makeTick` 很薄（只想念+重放最高 salience），没做真正的"无人时也在成长"。
- **同质化风险**：以上缺口让不同命在"她在成为谁/她的人生故事"上显得空、雷同。

## 2. 设计（分两层，先做对不可逆的）

### A 层（投影层，纯派生、可重算、**无版本变更**、低风险）——本轮先落
1. **修人生篇章，使其"活"**：① 选择改为**保留起点 + 最近**（始终能看到近况，随生活持续更新）；② 转折点来源**拓宽**到——确立的兴趣（"我渐渐着迷于 X"）、刻进心里的世界记忆（"读到…，记到现在"）、关系起止、刻骨记忆、确立的价值、成长里程碑。→ 叙事身份重新"持续更新"。
2. **`growth`（阅历/成长）**：由 年龄 + 关系数 + confirmed 价值数 + 兴趣数 + 记忆量 确定性合成一句"她活了多少、长成了什么形状"。
3. **`becoming`（我正在成为的我）**：由 top confirmed 价值 + top 兴趣 + 气质底色 合成的**演化中的独立自我**一句话。
4. **广播进"嘴"**：`growth/becoming` 进 `deriveWorkspace.selfFacts`（不含任何用户、纯她自己）→ 她说话带着"在成为谁"的连续自我 → 反同质化。
5. **展面**：资料页显示 `growth/becoming/兴趣`（脱敏，**不含 chapters**——篇章带用户名，仅 owner 后台可见）。

### B 层（折叠层，动 appraisal/状态、**需版本升级 + 严格测试**）——**已落（RECONSTRUCT_VERSION 15）**
1. **心智成熟度 `maturity`（持续变聪明，不违反"气质终生不变"）**✅：独立于先天气质的派生量 `RState.maturity`，**只在"有所学"的反思后**按衰减收益缓慢累积（渐近 1、不无界），**轻微调制情绪复原速度**（满成熟最多 +25%，气质仍是底色）。严守：① 上界小、确定性；② 不改写气质（maturity 是其上薄修正）；③ 出生=0、无学习不长 → 兼容。已测：累积/有界/确定性/出生为 0。
2. **长期心愿 `aspirations`（独立意志）**✅：从 confirmed 价值/兴趣/长期牵挂**确定性沉淀**"她想去的方向"（投影层），广播进"嘴"(selfFacts) + 资料页"她想去的方向" → 她说话带着自己的方向、不只反应。
3. **更厚的 DMN `makeTick`**：发现 `makeTick` 已按 salience 在**全部**当前记忆（含世界记忆）里选最显著的做 reconsolidation → 世界经验**本就在被巩固**、随心情重染。故本轮不动 makeTick（避免加重"每跳 reconsolidate 一条"的既有增长）。**遗留待办**：① 每跳 reconsolidate 同一条 → st.memory 无界增长（永生尺度的卫生问题，独立于本设计）；② 偶发"自发洞见"（连两条记忆成一句）仍可做。

## 3. 守的契约 / 防的老坑
- 契约①：以上全是**确定性折叠/投影**，模型只把 `growth/becoming/心愿` 说出口，绝不产出它们。
- 契约③：篇章/叙事**只读、不回写身份**；`renarrate` 绝不漂移价值。
- 反 v0.x：衰减/成熟度**投影或受界约束**、不无界；DMN 不空转刷永不收口的窗口（有限近窗、收敛）；**活不依赖模型**（廉价模型下她照样在长）。
- 隐私：篇章含人类用户名 → **绝不进公开资料页**；公开面只放脱敏的 growth/becoming/兴趣。

## 4. 本轮落地（A 层）验收
- 篇章随新经历**持续更新**、且能看到近况（测试：长序列下近期转折点出现在篇章里，不再只剩最早 8 条）。
- `growth/becoming` 确定性、可重算（V2 不破）；进资料页与"嘴"。
- 现有命轨迹**逐位不变**（A 层不动折叠 → stateHash 对既有事件不变）。
